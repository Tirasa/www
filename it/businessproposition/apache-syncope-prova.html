    Intenzionato a provare Apache Syncope?

    Il progetto Apache Syncope offre molti modi di valutarne le funzionalità, inclusa una demo online.

    In aggiunta a questi, Tirasa ne fornisce altri particolarmente utili per vedere Apache Syncope al lavoro, integrato in una soluzione IAM completa o già pronto per il Cloud.

    Immagine Docker

    Questa demo richiede Docker configurato: se non è stato fatto in precedenza, procedere con l'installazione.

    Innanzitutto, richiedere il link per il download; una volta scaricato, proseguire come segue:

    1. Decomprimere 
      gunzip syncopecasdemo.tar.gz
    2. Caricare in Docker 
      docker load < syncopecasdemo.tar
    3. Eseguire 
      docker run -d -p 8443:443 -p 8080:6080 \
  --name syncopecasdemo syncopecasdemo:2.0.4

    Infine, puntare il browser web preferito su https://localhost:8443 per accedere alla documentazione della demo, che fungerà da guida attraverso le funzionalità disponibili.

    Non è necessario preoccuparsi dell'allerta di sicurezza relativa al certificato del server: in caso di incertezza, seguire queste istruzioni.

    Juju

    Appassionato di cloud e delle sue infinite possibilità? Tirasa ha sviluppato una charm ed un bundle per Juju, la piattaforma di sviluppo cloud di Canonical, che consente di impostare e gestire una infrastruttura Apache Syncope con grande semplicità, e di poterla utilizzare in maniera transparente con i provider più diffusi: Amazon EC2, OpenStack, Microsoft Azure, HP, ...
