    Intenzionato a provare Apache Syncope?

    Il progetto Apache Syncope offre molti modi di valutarne le funzionalità, inclusa una demo online.

    In aggiunta a questi, Tirasa ne fornisce altri particolarmente utili per vedere Apache Syncope al lavoro, integrato in una soluzione IAM completa o già pronto per il Cloud.

    Virtual Machine

    Scarica, installa e avvia

    1. Scarica l'immagine per VirtualBox
    2. Avvia VirtualBox
    3. Importa SyncopeDemo.ova tramite VirtualBox-->File-->Import Appliance...
    4. Avvia la macchina virtuale SyncopeDemo
    5. Scegli il kernel di default all'avvio o aspetta alcuni secondi 
    6. Controlla che tutti i servizi si siano avviati correttamente 
    7. Inserisci le credenziali di accesso (User: root, Password: password)
    8. Prendi nota dell'ip della macchina virtuale (ifconfig eth0, ipvirtualmachine di seguito)
    9. Punta il browser favorito su http://ipvirtualmachine:8080/syncope-console
    10. Attenti alcuni istanti affinché le applicazioni completino l'installazione
    11. Inserisci le credenziali di accesso (User: admin, Password: password)
    12. Benvenuto nella console di amministrazione di Apache Syncope!

    Apache Syncope

    Oltre a quanto riportato sopra, altre informazioni:

    • Connector bundles directory: /var/tmp/syncope/bundles
    • Log directory: /var/tmp/syncope/log
    • Lo storage interno è configurato per PostgreSQL (nome del database, username e password sono tutti importati a syncope)

    OpenLDAP

    OpenLDAP è configurato come risorsa esterna di esempio allo scopo di fare provisioniing di utenti e ruoli via LDAP.

    • Host: ipvirtualmachine
    • Port: 389
    • Administrator: cn=admin,dc=syncope,dc=tirasa,dc=net
    • Password: password
    • Base Context: dc=syncope,dc=tirasa,dc=net

    In alternativa, è possibile accedere a http://ipvirtualmachine/phpldapadmin

    MySQL

    MySQL è configurato come risorsa esterna di esempio allo scopo di fare provisioning di utenti su un database relazionale.

    • User: root
    • Password: password

    La  console è all'indirizzo http://ipvirtualmachine/phpmyadmin

    Juju

    Appassionato di cloud e delle sue infinite possibilità? Tirasa ha sviluppato una charm ed un bundle per Juju, la piattaforma di sviluppo cloud di Canonical, che consente di impostare e gestire una infrastruttura Apache Syncope con grande semplicità, e di poterla utilizzare in maniera transparente con i provider più diffusi: Amazon EC2, OpenStack, Microsoft Azure, HP, ...

    Apache Syncope

    Abbiamo concepito Syncope perché esperti di IdM e appassionati di Open Source. L'idea era quella di creare un Identity Manager Open Source agile, leggero ed incisivo in grado di soddisfare i requisiti degli ambienti enterprise.

    Abbiamo fondato Tirasa con l'obiettivo di acceleare lo sviluppo e la diffusione di Syncope IdM.

    Nel Febbraio 2012 abbiamo donato Syncope IdM all'Apache Software Foundation.
